GetOrInsertTexture(AssetLocation, out Int32, out TextureAtlasPosition, CreateTextureDelegate, Single)

Same as InsertTexture(IBitmap, out Int32, out TextureAtlasPosition, Single) but this method remembers the inserted texure, which you can access using capi.TextureAtlas[path] A subsequent call to this method will update the texture, but retain the same texPos. Also a run-time texture reload will reload this texture automatically.

Declaration
